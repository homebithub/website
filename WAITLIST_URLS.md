## Notes for Marketing Team

- **Default Profile**: If no `profile` parameter is provided, the page defaults to `household` view
- **Service Selection**: Each page allows users to select multiple related services from the full service list
- **Contact Form**: All pages include a contact form with searchable country phone code dropdown (all countries supported)
- **Share Functionality**: After joining the waitlist, users can share their unique referral link via native share or copy to clipboard
- **WhatsApp Opt-in**: Users can opt in to receive WhatsApp updates when the service onboarding opens
- **Concern Selection**: Users can select their biggest concerns (Trust, Speed, Experience, Affordability, Education, Driver's License, Professionally Trained)
